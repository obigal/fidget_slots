# Fidget Slots
Simple slot machine game for the Arduboy.

A          - Spin/New Game.

B          - Toggle autospin.

UP/DOWN    - Bet increase/decrease (1 credit per line).

LEFT       - Toggle sound.

RIGHT      - Toggle auto-restart with autospin (play forever).
